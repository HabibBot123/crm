3. Dashboard Layout
The layout wraps all dashboard pages with:


layout.tsx
Lines 1-22
import { DashboardSidebar } from "@/components/dashboard/dashboard-sidebar"
import { MobileNav } from "@/components/dashboard/mobile-nav"
import { OrganizationProvider } from "@/components/providers/organization-provider"
import { DashboardAccessGate } from "@/components/dashboard/dashboard-access-gate"
export default function DashboardLayout({ children }: { children: React.ReactNode }) {
  return (
    <OrganizationProvider>
      <DashboardAccessGate>
        <div className="flex min-h-screen bg-background">
          <DashboardSidebar />
          <div className="flex flex-1 flex-col">
            <MobileNav />
            <main className="flex-1">
              {children}
            </main>
          </div>
        </div>
      </DashboardAccessGate>
    </OrganizationProvider>
  )
}
OrganizationProvider — multi-org context
DashboardAccessGate — auth/permission guard
DashboardSidebar — collapsible left sidebar (desktop only, with OrgSwitcher and ThemeToggle)
MobileNav — top nav bar for mobile
Sidebar nav items (in order): Overview, Products, Offers, Clients, Coaching, Content, Branding, Team, Settings

4. Main Dashboard Page (/dashboard)
Displays an analytics overview using a Supabase RPC call (dashboard_overview):

KPI grid (4 cards): Total Revenue, Total Clients, Total Sales, Conversion Rate — via the custom StatCard component
CTA banner: Shows a prompt when there are unassigned coaching packs
Recent Sales panel: Avatar + client name + offer title + amount
Top Products panel: Product type icon + title + enrollment count + Badge variant
5. Color Palette / Design System
The project uses Tailwind CSS v4 with OKLCH color tokens defined in globals.css. It supports light/dark modes via the class strategy.

Light Mode
Token	Value	Approximate Color
--background	oklch(0.985 0 0)	Near-white
--foreground	oklch(0.13 0 0)	Near-black
--card	oklch(1 0 0)	Pure white
--primary	oklch(0.45 0.18 250)	Deep blue/indigo
--primary-foreground	oklch(0.99 0 0)	White
--secondary	oklch(0.96 0.005 250)	Very light blue-gray
--muted	oklch(0.955 0 0)	Light gray
--muted-foreground	oklch(0.50 0 0)	Mid gray
--accent	oklch(0.70 0.15 170)	Teal/green
--destructive	oklch(0.55 0.22 27)	Red/orange
--success	oklch(0.60 0.18 145)	Green
--warning	oklch(0.75 0.15 65)	Amber/yellow
--border	oklch(0.91 0 0)	Light gray
--sidebar	oklch(0.98 0 0)	Off-white
--radius	0.75rem	Rounded corners
Dark Mode
Background shifts to oklch(0.13 0 0) (very dark), cards to oklch(0.17 0 0), primary becomes slightly brighter blue oklch(0.55 0.20 250).

Typography
Body font: Inter (Google Fonts, variable --font-inter)
Display font: Space Grotesk (Google Fonts, variable --font-space-grotesk)
Used via the font-display utility class on headings and stat values
