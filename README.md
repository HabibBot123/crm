# CoachStack — CRM for coaches

Application CRM pour tout type de coach (éducation, business, life, sport, executive, etc.) : gestion clients, ventes, offres, contenu, équipe, suivi de programmes, etc.

## Prérequis

- **Node.js** 18.x ou supérieur (recommandé : LTS)
- **pnpm** (recommandé) ou npm / yarn

## Installation des dépendances

À la racine du projet :

```bash
pnpm install
```

Avec npm :

```bash
npm install
```

Avec yarn :

```bash
yarn install
```

## Lancer le projet en développement

```bash
pnpm dev
```

Avec npm :

```bash
npm run dev
```

L’application est disponible sur **http://localhost:3000**.

## Autres commandes

- **Build (production)** : `pnpm build`
- **Démarrer en production** : `pnpm start` (après un `pnpm build`)
- **Linter** : `pnpm lint`
